docs:
  Folder containing documentation to describe the basic functionality of the 
  FDC main sketches
fdc_main:
  Folder containing the Arduino sketch used in the FPT Driverless Car. This 
  version run without PID
fdc_main_w_pid:
  Folder containing the Arduino sketch used in the FPT Driverless Car. This 
  version run with PID

On the FPT Driverless Car, the Arduino's purpose is to serve as a bridge 
between the Jetson TK1 to the sonar sensor, the button-with-LED, and the 
ESC, which controls the engine.
